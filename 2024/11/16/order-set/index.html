底线模式

在命令模式下输入 : 即可进入底线命令模式，用于执行文件相关操作。

命令 说明
:w 保存文件
:q 退出编译器
:wq 保存并退出
q! 强制退出

命令模式

刚启动Vim时，进入的就是命令模式

命令 说明
i 进入输入模式
x 删除光标所在字符
dd 剪切当前行
yy 复制当前行
p 粘贴到光标下方
P 粘贴到光标上方
u 撤销
Ctrl+r 撤销上一次撤销
o 在当前行上方插入新行
O 在当前行下方插入新行

而输入模式与平时的输入没有什么不同，在此省略。
